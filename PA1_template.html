      if(activity1$date[i] == "Saturday" | activity1$date[i] == "Sunday") {

  1. Make a panel plot containing a time series plot (i.e. type = “l”) of the 5-minute interval (x-axis) and the average number of steps taken, averaged across all weekday days or weekend days (y-axis). The plot should look something like the following, which was creating using simulated data:
weekdays <- subset(activity1, date == "weekday")
weekends <- subset(activity1, date == "weekend")

weekends <- tapply(weekends$steps, weekends$interval, sum)

The plots show that people get up earlier on weekdays, and sleep in on weekends. During the day on weekdays, people presumably sit more while on weekends, people walk more spread throughout the day.
